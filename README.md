# DRL Homework 2: Cliff Walking

This repository contains the implementation of the Cliff Walking environment from Reinforcement Learning: An Introduction (Sutton and Barto).

## Files
- `hw2.py`: Python script implementing Q-learning and SARSA algorithms.
- `Figure_1.png`: Result plot comparing the performance of SARSA and Q-learning.
- `作業2.md`: Assignment documentation and report in Markdown format.

## Requirements
- `numpy`
- `matplotlib`

## Running the code
Run the Python script to train the agents and generate the policy comparison and reward plot:
```bash
python hw2.py
```
